# 📋 Prerequisites

Before you start building with PyWebApp Native, ensure your development environment is correctly set up.

## 🐍 Python Environment
PyWebApp is built for modern Python.
- **Python 3.9 or newer** is required.
- **pip** (Python package manager) should be installed and updated.
- (Recommended) Use a virtual environment (`venv`) to keep your project dependencies clean.

## ⚛️ Frontend Environment
Since the UI is built with standard Web technologies:
- **Node.js (v18 or newer)** is required.
- **npm** (usually bundled with Node.js) or **yarn**.
- Basic knowledge of **React** (or your chosen JS framework).

## 📱 Mobile Development (Android)
To build and run Android applications:
- **Android Studio** (Koala or newer recommended).
- **JDK 17** (Ensure your `JAVA_HOME` points to JDK 17).
- **Android SDK 34** or higher.
- A physical Android device or an **x86_64 Emulator**.

## 💻 Desktop Development
- **Windows / Linux / macOS**: Standard OS build tools are usually sufficient.
- On Windows, ensure you have the **Microsoft Visual C++ Redistributable** installed.

---

## 🛠️ Verification
Once you have the basics, you can verify your environment by running:

```bash
# Check Python
python --version

# Check Node
node --version

# Check NPM
npm --version
```

If everything looks good, you're ready to [Get Started!](./getting-started) 🚀
