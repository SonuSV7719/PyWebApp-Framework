# 📱 Android Elite Guide

PyWebApp Native transforms your web logic into high-performance Android applications.

## 🚀 Building for Android
Run the following command to generate your signed APK:
```bash
pywebapp build-android
```

## 📸 Native Features
Your app comes pre-configured with:
- **Camera Access:** Automatic permission handling.
- **File System:** Unified picker for PDF, Images, and Data.
- **Hardware Acceleration:** Enabled by default for 60fps UI.

## 🛡️ Production Security
The framework automatically applies **ProGuard** rules to shrink and obfuscate your code, making it secure against reverse-engineering.
